# Hotel Booking App

A React.js project that fetches hotel data from an API and displays hotel cards.

## Features

- Hotel image
- Hotel name
- Description
- Location
- Rating
- Price per night
- Wishlist button
- Previous / Next pagination
- Responsive design

## Tech Stack

- React.js
- CSS
- Fetch API

## API

https://demohotelsapi.pythonanywhere.com/hotels/
